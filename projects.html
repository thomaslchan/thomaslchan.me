Projects


Image Compression

The program accepts a PPM image as input and compresses the image. The compression process begins by converting pixel color values from the RGB format to the component video format through a discrete cosine transform. It then packs the resulting values into binary words. The program can decompress images through the inverse procedure.

This project was part of the coursework for COMP40: Machine Architecture and Assembly Language, assigned with the intention of understanding floating point arithmetic in C. I partnered with Steven Song.


Network Scan Alarm

This program can perform live sniffing of a network (default eth0) or analyze packet data given a pcap file. The alarm can detect NULL, FIN, Xmas, and Nikto scans; usernames/passwords sent via HTTP Basic Authentication or FTP; and identify source IPs from Russia or Facebook. This was written in Python, using the scapy and pcapy libraries.

This project was assigned by COMP116: Introduction to Computer Security.
