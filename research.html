Research


Blavatnik Institute

I helped developed a tool for single-cell experiment data, vitessce.io. I focused on the data processing pipeline for high-resolution microscopy images. It converts HDF5 files to an OME-TIFF format and then creates image tiles to allow for easy rendering when zooming.

This was developed under Nils Gehlenborg as part of the 2019 Summer Institute in Biomedical Informatics (SIBMI) program at Harvard Medical School. It was written using Python, bash scripting, and AWS S3 command line tools.

Boyce Thompson Institute

I created a marker-pedgree visualization tool for CassavaBase. The tool is designed for genomics-assisted breeding of cassava. Breeders can select crop accessions and SNPs, and the tool will render the data as a pedigree with each individual's SNP values displayed adjacent to it.

This work was done under Lukas Mueller at The Boyce Thompson Institute for an REU during the summer of 2018. It was written with Perl, HTML, JavaScript (D3.js), and PostgreSQL.
