Research


Boyce Thompson Institute

In the summer of 2018, I worked under Lukas Mueller. I created a marker-pedgree visualization tool for CassavaBase. The tool is designed for genomics-assisted breeding of cassava. Breeders can select crop accessions and SNPs, and the tool will render the data as a pedigree with each individual's SNP values displayed adjecent to it. It was written with Perl, JavaScript (D3), and PostgreSQL.

Blavatnik Institute

This summer, I'm working with Nils Gehlenborg at Harvard Medical School's Department of Biomedical Informatics. I'm working on a tool for single cell experiment data, vitessce.io.

SongWorm

This was written during Tufts Polyhack 2017. It takes text from the user and generates sounds, mapping each letter to one of five notes. Essentially, it's a text-to-sound converter. This was a collaborative effort with four other Tufts students. It was implemented with HTML, CSS, and Javascript.
